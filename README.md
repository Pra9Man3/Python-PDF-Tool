# AnnexPDF – Smart PDF Utility for Professionals

AnnexPDF is a Python-powered desktop tool designed to help professionals prepare clean, compliant, and portal-friendly PDF documents for regulatory submissions such as GST, Income Tax, ROC, and other filings.

It simplifies PDF handling by automating repetitive tasks like merging, labeling, numbering, and optimizing documents for smooth portal uploads.

---

## Key Features

✅ Merge multiple PDFs into a single document  
✅ Add Annexure labels automatically  
✅ Page numbering support  
✅ Custom label positioning (X/Y controls)  
✅ PDF compression & optimization  
✅ Portal-friendly output generation  
✅ Simple desktop GUI for easy use
